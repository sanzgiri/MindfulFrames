# MindfulFrames for iOS

This directory contains the native SwiftUI version of MindfulFrames. It is intentionally separate from the web application while sharing the same GitHub repository.

## Open on a Mac

1. Clone or pull the repository.
2. Open `MindfulFrames-iOS/MindfulFrames.xcodeproj` in Xcode.
3. Select the `MindfulFrames` scheme and an iPhone simulator or registered device.
4. Build and run.

The app uses bundle identifier `com.sanzgiri.mindfulframes`. Photos, journal entries, course progress, settings, and reminder preferences are stored locally on the device.
